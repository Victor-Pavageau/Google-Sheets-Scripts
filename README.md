# Google Sheets Scripts

A Google Apps Script implementation for automating tasks in Google Sheets, featuring a random movie selector and a week countdown calculator.

## About

This project is a Google Apps Script implementation designed for personal use with Google Sheets. The repository contains a `main.gs` file, which includes two primary scripts:

- RANDOM MOVIE Selector: This script picks a random cell within a specified range, ideal for selecting a random movie from a predefined list.
- WEEK COUNT Script: This script calculates the number of weeks remaining between two given dates, providing a quick and efficient way to manage time-based tasks or deadlines.

The project was developed as a personal initiative to automate and simplify recurring tasks in Google Sheets. By leveraging Google Apps Script, it showcases practical use cases for lightweight automation in everyday scenarios, enhancing productivity and organization.
